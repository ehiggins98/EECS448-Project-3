    func Forward ¶ 

    func Forward(w http.ResponseWriter, r *http.Request)

    Accepts a request from the user and forwards it to the appropriate worker. If the server doesn't respond, it will be reset and rebooted asychronously. w is the response output and r is the input request.

    func GetExecutor ¶ 

    func GetExecutor() (string, error)

    Gets the IP address of an available execution server from workers. Returns a string containing the VM's ID, or an error if no workers are available.

    func GetIP ¶ 

    func GetIP(id string) (string, error)

    Get the internal IP address of a given worker instance. The given id should be the name of the instance, such as "worker-1".

    func Init ¶ 

    func Init()

    Authenticate with Google Cloud using JWT and the existing service account. This gets the data from `key.json`.

    func MakeRequest ¶ 

    func MakeRequest(server string, body string) *http.Request

    Builds an HTTP request with properties 

    URL: http://{server}
Method: POST
Headers: {Content-Type: "application/json"}

    func Reset ¶ 

    func Reset(id string) error

    Reset a worker instance. This resets the VM to its initial state, stops the instance, and restarts it. The given Id is the name of the instance (for example, "worker-1"). Returns an error if one occurs in the reset process.

    func TimeMillis ¶ 

    func TimeMillis() int64

    Gets the current system time in milliseconds.

    type Data ¶

    A struct used for storing the necessary data from the service account input file. The file contains more information, but the only information needed for authentication are the client email and private key.

    ClientEmail string `json:"client_email"`
    PrivateKey  string `json:"private_key"`
