Input¶

The input pipeline for the character classifier.

input.base_process(tensor)¶

Performs only normalization to process the given image and label.

Parameters:tensor (tf.Tensor) – A tensor containing the image and label to process.
Returns:The normalized image and its corresponding label.
input.eval_input_fn()¶

Provides input images and labels for the evaluation procedure.

Returns:Operations by which the dataset can be obtained.
input.get_image_and_label(tensor)¶

Extracts the label and image from the given tensor.

Parameters:tensor (tf.Tensor) – The Tensor from which to extract the label and image.
Returns:The image and label.
input.normalize(image)¶

Normalizes the given image by dividing by the max value (255) and subtracting the mean.

Parameters:image (tf.Tensor) – The image to normalize.
Returns:The normalized image.
input.process(tensor)¶

Performs full processing on the given image and label for use during training. This includes randomly cropping the image, randomly scaling the image, and randomly rotating the image to an angle between 0 and 10 degrees.

Parameters:tensor (tf.Tensor) – A tensor containing the image and label to process.
Returns:The processed tensor and its corresponding label.
input.process_dataset(dataset, process_fn)¶

Creates operations for processing batches of data.

Parameters:
  • dataset (tf.data.Dataset) – The dataset to process.
  • process_fn (function) – The function to apply to elements of the dataset.
Returns:

Operations for processing the batches of data.
input.test_input_fn()¶

Provides input images and labels for the testing procedure.

Returns:Operations by which the dataset can be obtained.
input.train_input_fn()¶

Provides input images and labels for the training procedure.

Returns:Operations by which the dataset can be obtained.
