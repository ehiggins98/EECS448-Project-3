Server¶

A Flask server that takes an image, processes text in it, and returns the result.

server.aggregate(probabilities, chars)¶

Aggregates several probabilities into a value indicating the likeliness of the given sequence of characters.

Parameters:
  • probabilities ([float]) – The list of probabilities to aggregate.
  • chars ([char]) – The list of characters corresponding to the given probabilities, such that the character at i corresponds to the `i`th probability.
Returns:

The aggregated probability.
server.decode(index)¶

Decodes a character class value to its corresponding character.

Parameters:index (int) – The character class value to decode.
Returns:The character corresponding to the given class value, or None if the given class is invalid.
server.encode(char)¶

Encodes a character as the corresponding class value.

server.find_probs(start, offset, probabilities)¶

Find the offset-th valid probability after start.

Parameters:
  • start (int) – Starts searching for probabilities at the index after this value.
  • offset (int) – The offset to return. For example, if offset=3, the function will return the 3rd valid probability vector after start.
  • probabilities (np.ndarray) – The probability matrix.
Returns:

The offset-th valid probability vector after start.
server.list_from_regex(regex_str)¶

Creates a list of valid characters given a regular expression.

Parameters:regex_str (string) – The regex for which to get a list.
Returns:A list of characters accepted by the given regex.
server.lookahead(context, prob1, prob2, prob3, prob4)¶

Finds the character that results in the highest probability among it and the next character.

Parameters:
  • context (Scope) – The current context in the code.
  • prob1 (np.ndarray) – The probability distribution associated with the current character.
  • prob2 (np.ndarray) – The probability distribution associated with the next character.
  • prob3 (np.ndarray) – Unused
  • prob4 (np.ndarray) – Unused
Returns:

The current character that maximizes probability over the next two.
server.process_image()¶

Receives an image from an HTTP request and processes the text. The request must have one field titled “file,” for which the value is the bnary image.

server.runTests()¶

Runs the test suite and returns the result through an HTTP request.

server.sum_similar(probabilities, character)¶

Sums probabilities over similar characters. For example, if character == ‘c’, then this might sum probabilities over ‘c’, ‘C’, and ‘<’.

Parameters:
  • probabilities (np.ndarray) – The probabilities to sum.
  • character (char) – The character for which to sum similar characters’ probabilities.
Returns:

The summed probability.
