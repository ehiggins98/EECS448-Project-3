Test Classifier Manual¶

A script for manually testing the entire character classification and context parsing algorithm. When the user enters a character, it fetches a character of that class from the dataset, shows it to the user, and passes it to the context parser as the actual software would.

test_classifier_manual.decode(index)¶

Decodes a class encoding as its character.

Parameters:index (int) – The class encoding to decode.
Returns:The decoded character.
test_classifier_manual.encode(char)¶

Encodes a character as its class mapping.

Parameters:char (char) – The character to encode.
Returns:The character class mapping for char.
test_classifier_manual.list_from_regex(regex_str)¶

Gets a list of characters accepted by the given regular expression.

Parameters:regex_str (string) – The regex to consider.
Returns:A list of characters accepted by the regex_str.
